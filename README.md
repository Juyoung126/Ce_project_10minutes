# Ce_project_10minutes

가천대학교 2023-2 졸업프로젝트

1. 프로젝트 소개
 YOLOv5 모델을 활용해 쓰레기 탐지 및 분리배출 방법을 제공하는 서비스를 개발했습니다. 사용자 주변의 공공 쓰레기통 위치를 안내하는 지도 기능도 포함하여, 분리배출의 중요성을 알리고 지속 가능한 소비 문화를 촉진하는 환경 친화적인 솔루션을 구현했습니다.
